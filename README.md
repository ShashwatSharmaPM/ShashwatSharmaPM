**Senior Product Manager** based in Berlin. 9 years building API platforms, integration ecosystems, and SaaS products across fintech, logistics, and traveltech. I specialize in creating smart integrations, tracking and analyzing user behavior, and iterating on product experiences to drive measurable impact.

## What I Do

I build products where the "user" is often another system, another team, or another company's API. That means thinking about developer experience, integration scalability, and platform reliability, not just UI flows and feature requests.

Some of the problems I've worked on:

- Breaking a monolith into 50+ microservice APIs and launching a modular API marketplace that opened up a new market segment
- Building a 0-to-1 browser plugin with an AI personalization layer that solved a data black-box problem no one else in the value chain could crack
- Taking an open banking payments platform from regulatory whitepaper to $17M ARR across 7 global clients
- Migrating legacy SOAP integrations to REST, cutting latency by 80% and reducing onboarding timelines from months to weeks
- Designing self-service RBAC for a multi-tenant platform serving 100K+ users across retail, logistics, and distribution

## How I Work

**Discovery**: I talk to implementation teams, account managers, and customers, not just end users. The best product insights often come from the people debugging production issues at midnight.

**Artifacts**: PRDs, technical narratives, Figma collaboration, demo docs with click-through walkthroughs. I create whatever the team needs to ship with confidence, not whatever the process demands.

**Engineering collaboration**: Daily syncs with EMs when projects are hot. Weekly refinements. I debug issues even if I can't fix them. I know enough about the code to have a real conversation about trade-offs, and enough humility to defer on system design.

**Prioritization**: RICE when it works, business case narratives when it doesn't. I've said no to leadership with a backlog in hand, and I've convinced leadership to pause feature work for a year to invest in platform infrastructure. Both required the same thing: clear data and a story that connects to revenue.
